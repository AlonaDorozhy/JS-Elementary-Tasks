Palindrome

Input Parameters: Number

Lucky tickets

There are 2 ways to count lucky tickets; you need to determine programmatically which option for counting lucky tickets will give them a greater number at a given interval

Input parameters: context object with fields min and max

Output: an object with information about the winning method and the number of lucky tickets for each calculation method.

Numerical sequence

Input parameters: length and minimum square value

Output: a string with a series of numbers
